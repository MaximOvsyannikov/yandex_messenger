# yandex_messenger

TS + handlebars
Block components, EventBus, Lint

## Installation

```bash
$ npm install
```

## Running Development Mode

```bash
npm run dev
```

## Running build

```bash
npm run build
```

## Running express on localhost:3000

```bash
npm run start
```

## Lint check

```bash
npm run lint
```
